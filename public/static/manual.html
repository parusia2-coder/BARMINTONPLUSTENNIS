1. 시스템 개요

웹 기반 배드민턴 대회 운영 시스템입니다. 인터넷 브라우저만 있으면 별도 앱 설치 없이 모든 장비에서 사용할 수 있습니다.

주요 화면 4가지

화면URL대상용도
관리 메인 관리 / 운영 본부 대회 관리 전체 (참가자, 종목, 경기, 결과)
코트 점수판 /court?tid=1&court=번호 코트 태블릿 전체화면 터치 점수 입력
통계 대시보드 /dashboard?tid=1 전광판 / 모니터 전체 현황 실시간 표시 (30초 자동갱신)
참가자 내 경기 /my?tid=1 참가자 스마트폰 본인 경기 일정/결과 조회

[인터넷 회선] │ ┌─────────┴─────────┐ │ Wi-Fi 공유기 │ ← 5GHz 듀얼밴드 권장 │ (코트 중앙 설치) │ └─────────┬─────────┘ ┌──────┬───────┼───────┬───────┬──────┐ 코트1 코트2 코트3 코트4 코트5 코트6 태블릿 태블릿 태블릿 태블릿 태블릿 태블릿 │ ┌────┼────┐ 본부 관중 참가자 노트북 모니터 스마트폰
⚠️ 네트워크 주의사항
  • 체육관은 Wi-Fi 사각지대가 많음 → 5GHz 듀얼밴드 공유기 사용
  • 코트 태블릿은 5GHz, 참가자 스마트폰은 2.4GHz로 대역 분리 권장
  • 체육관이 넓으면 공유기 2대 (본부 + 코트 중앙)
  • 핫스팟 백업: 공유기 장애 대비 스마트폰 테더링 준비

문제 상황원인해결 방법

점수가 반영 안 됨 Wi-Fi 끊김 오프라인 경고 확인 → Wi-Fi 재연결 → 새로고침

태블릿 배터리 방전 장시간 사용 보조배터리 연결 또는 예비 태블릿에 같은 URL 접속

대시보드 갱신 안 됨 브라우저 탭 비활성 탭 클릭하여 활성화 또는 새로고침

9. 효율적 운영 팁

✅ 코트 자율 운영

locked=1&autonext=true 설정하면 코트 기록원만 있으면 본부 개입 없이 경기가 자동 순환됩니다.

✅ 경기 호출 타이밍

대시보드의 코트 현황에서 빈 코트가 보이면 즉시 다음 경기 팀을 방송 호출합니다.

✅ 결선 타이밍

모든 종목 예선 80%+ 완료 시점부터 완료된 종목부터 순차 결선 생성이 가능합니다.

✅ 관중 실시간 관람

코트 태블릿의 QR 버튼으로 관중이 스마트폰에서 해당 코트 실시간 점수를 볼 수 있습니다 (mode=view).

✅ 중간 점검

50% 진행률 시점에서 종목별 진행 상황을 확인하고 지연 종목을 우선 배정합니다.

✅ 분쟁 대비

모든 경기는 승/패팀 서명으로 확인됩니다. 중요 경기는 서명 화면 캡처를 권장합니다.
