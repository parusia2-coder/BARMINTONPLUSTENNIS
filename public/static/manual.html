1. 시스템 개요

웹 기반 대회 운영 시스템입니다. 인터넷 브라우저만 있으면 별도 앱 설치 없이 모든 장비(태블릿, 노트북, 스마트폰)에서 사용할 수 있습니다.

🏸 배드민턴 / 🎾 테니스 2종목 지원

빌드 시 환경변수(SPORT=badminton 또는 SPORT=tennis)로 종목을 선택하면, UI 테마·용어·점수 규칙·종목 구성이 해당 스포츠에 맞게 자동 전환됩니다.

2. STEP 1 — 대회 만들기

홈 화면에서 "새 대회 만들기" 버튼을 클릭하면 대회 생성 폼이 열립니다.

입력 항목 설명

항목필수 여부기본값설명
🏸🎾 종목 선택 필수 배드민턴 배드민턴: 남복(MD) · 여복(WD) · 혼복(XD) 종목 지원. 점수 단위 '점', 기본 목표 25점
테니스: 남단(MS) · 여단(WS) · 남복(MD) · 여복(WD) · 혼복(XD) 종목 지원. 점수 단위 '게임', 기본 목표 8게임, 체인지오버/듀스 규칙 설정 가능

※ 종목에 따라 UI 테마(색상), 용어, 카테고리, 채점 규칙이 자동 전환됩니다.
※ 하나의 시스템에서 배드민턴 대회와 테니스 대회를 동시에 운영할 수 있습니다.
대회명 필수 - 예: "2026 봄 배드민턴 대회", "제5회 테니스 클럽 챔피언십"
설명 선택 - 대회 안내 사항 (장소, 일시, 참가비 등)
대회 방식 필수 KDK KDK (랜덤 대진): 팀당 N경기 랜덤 대진 — 가장 많이 사용
풀리그: 모든 팀이 서로 한 번씩 대전
토너먼트: 싱글 엘리미네이션 (지면 탈락)
코트 수 필수 6 실제 사용할 코트 수 (1~20). 경기 배정 시 이 수만큼 동시 진행
팀당 경기 수 선택 4 KDK 방식일 때 각 팀이 최소 몇 경기를 하는지 설정
급수합병 기준 선택 4 종목의 참가팀이 이 수 미만이면 인접 급수와 자동 합병
관리자 비밀번호 필수 - 대회 관리 기능 접근용. 참가자 등록/수정, 경기 관리 등에 필요
✅ 대회 생성 후
  • 대회 카드가 홈 화면 목록에 나타납니다 (🏸 배드민턴 또는 🎾 테니스 뱃지로 종목 구분)
  • 대회 카드를 클릭하면 대회 상세 페이지로 이동 (해당 종목의 테마/색상이 자동 적용)
  • 상세 페이지에서 "관리자" 버튼 → 비밀번호 입력하면 관리자 모드 활성화
  • 관리자 모드에서만 참가자 등록, 종목 생성, 경기 관리 등이 가능
💡 관리자 세션 유지

한번 인증하면 브라우저 localStorage에 저장되어, 페이지 새로고침/재접속해도 관리자 모드가 유지됩니다. 여러 대회를 동시에 관리할 수도 있습니다.

4. STEP 3 — 종목 생성

대회 상세 → "종목/팀" 탭 → 관리자 모드에서 생성합니다.

4-1. 개별 종목 추가

"종목 추가" 패널에서 아래 3가지를 선택합니다.

선택 항목옵션설명
종류 남자복식 / 여자복식 / 혼합복식
(테니스: 남단·여단도 가능)		 성별 기반으로 참가자 자동 필터링
연령대 오픈(전연령) / 20대 / 30대 / 40대 / 50대 / 55대 / 60대 "오픈"이면 전 연령 참가 가능
급수 전체 / S / A / B / C / D / E "전체"이면 모든 급수 참가 가능

→ "종목 추가" 버튼 클릭하면 종목이 생성됩니다 (예: "남자복식 50대 B급")

4-2. 종목 일괄 생성 (다중선택)

"일괄 생성" 버튼을 클릭하면 4단계 선택 모달이 열립니다.

STEP 1 → 종목 유형 선택 STEP 2 → 연령대 선택 STEP 3 → 급수 선택 STEP 4 → 팀편성 방식 ──────────────── ──────────────── ──────────────── ──────────────── ☑ 남자복식 ☑ 오픈 (전연령) ● 전체 급수 ● 클럽 우선 ☑ 여자복식 ☑ 50대 (12명) ○ 개별 급수 ○ 급수 매칭 ☐ 혼합복식 ☑ 55대 (8명) ☐ S ☐ A ☐ B ... ○ 랜덤 ☐ 40대 (23명) ※ 각 연령대의 참가자 수가 표시됩니다
✅ 일괄 생성 예시

남복 + 여복 선택 → 50대 + 55대 선택 → 전체급수 → 클럽 우선

→ 자동으로 4개 종목 생성: 남자복식 50대 전체, 남자복식 55대 전체, 여자복식 50대 전체, 여자복식 55대 전체

→ 각 종목별 자동 팀편성까지 한번에 완료!

4-3. 급수 합병

참가 인원이 적은 종목은 인접 급수와 합병하여 운영할 수 있습니다.

기능설명
자동 합병 체크 "급수합병 체크" 버튼 → 설정된 기준(기본 4팀) 미만 종목을 자동 감지하여 합병 제안
수동 합병 "수동 합병" 버튼 → 합병할 종목들을 직접 선택하여 합병
합병 취소 합병된 종목 카드의 "합병취소" 버튼 → 원래 종목으로 분리
⚠️ 일괄 삭제 기능
  • "조편성 일괄삭제": 종목은 유지하되 팀/경기/순위만 전체 삭제 (재편성 시 유용)
  • "종목 전체삭제": 종목 자체를 모두 삭제 (처음부터 다시 시작)
  • 두 기능 모두 이중 확인 대화상자가 표시됩니다

5. STEP 4 — 조편성 (팀 배정)

종목/팀 탭에서 "조편성 옵션" 버튼을 클릭하면 설정 모달이 열립니다.

5-1. 조편성 옵션

1단계: 팀 편성 방식

방식설명적합한 상황
같은 클럽 우선
(권장)		 같은 소속 클럽 멤버끼리 먼저 매칭 → 남은 인원은 급수 순 매칭 클럽 소속 참가자가 많은 대회
같은 급수 매칭 클럽 무관, 같은 급수끼리 우선 매칭 (급수 밸런스 중시) 실력 균형을 중요시하는 대회
완전 랜덤 클럽·급수 무관 랜덤 매칭 친선 대회, 레크리에이션

2단계: 조(그룹) 배정

옵션설명
조 배정 실행팀 편성 후 자동으로 조 배정까지 진행 (체크 해제 시 팀만 편성)
조당 팀 수기본 5 (4~5팀 풀리그 권장). 조당 팀 수가 많으면 경기 수 ↑
같은 클럽 다른 조 배정같은 클럽 팀끼리 다른 조에 배정 (조별 리그 시 클럽 내 대결 최소화)

5-2. 미리보기 & 실행

1
옵션 설정 후 "미리보기" 버튼 → 편성 결과 미리 확인 (종목별 팀 수, 조 배정 현황)
2
결과가 마음에 들면 "편성 실행" 버튼 → 실제 데이터에 반영
3
종목 카드에 팀 수 배지 표시. "팀 목록 보기"를 클릭하면 상세 팀 구성 확인 가능
💡 수동 팀 등록도 가능

각 종목 카드의 "팀 등록" 버튼으로 팀을 수동으로 등록할 수 있습니다. 1번 선수와 2번 선수를 직접 선택하여 팀을 구성합니다. (단식은 1번 선수만 선택)

6. STEP 5 — 대진표 생성

종목/팀 탭에서 "대진표 옵션" 버튼을 클릭합니다.

6-1. 대진 방식 선택

방식설명적합한 상황
자동 결정
권장		 팀 수와 조 배정 여부에 따라 자동으로 최적 방식 결정
• 조 배정 있으면 → 조별 리그 (같은 조끼리 풀리그)
• 5팀 이하 → 풀리그
• 그 외 → KDK (랜덤 대진) 		대부분의 대회

풀리그 모든 팀이 다른 모든 팀과 한 번씩 대전 소규모 종목 (5~6팀 이하)
토너먼트 지면 탈락, 결승까지 승자끼리 대전 결선, 빠른 결정

6-2. 대진 옵션 & 대상 종목

옵션설명

1
방식 · 옵션 · 대상 종목을 설정
2
"대진표 생성" 버튼 클릭 → 확인 대화상자 (⚠️ 기존 경기 데이터 초기화됨)
3

7. STEP 6 — 경기 운영 (본부)

대회 상세 → "경기" 탭 → 관리자 모드에서 운영합니다.

7-1. 경기 시작

🎯 경기 시작 흐름
경기 카드 (대기 상태) │ ├─ "시작" 버튼 클릭 │ │ │ ├─ 경기 상태 → "진행중"으로 변경 │ ├─ 코트 점수판에 경기 정보 표시 │ └─ 🔔 해당 선수에게 푸시 알림 자동 발송 │ ("경기 시작!" + 코트번호 + 상대팀) │ └─ 코트에서 점수 입력 → 목표점 도달 시 자동 완료

경기 탭에는 각 경기가 카드 형태로 표시됩니다. 각 카드에 라운드, 코트 번호, 팀명, 현재 점수, 상태가 표시됩니다.

경기 상태색상의미가능한 작업
대기회색아직 시작 안 됨"시작" 버튼으로 경기 시작
진행중초록현재 코트에서 진행 중점수 입력/수정, 코트 변경
완료파란경기 종료점수 수정, 리셋

7-2. 점수 입력 & 수정

1
경기 카드의 "점수" 버튼 클릭 → 점수 입력 모달 표시
2
팀1 / 팀2의 세트 점수 입력 (세트 1~3까지 지원)
3
"저장" 버튼 → 점수 반영 + 승자 자동 판정
⚠️ 완료된 경기 수정 시

이미 완료된 경기의 점수를 수정할 때는 주황색 경고 배너가 표시됩니다. 수정 내역은 감사 로그(Audit Log)에 자동 기록됩니다.

7-3. 경기 리셋 · 코트 재배정

기능버튼설명
경기 리셋 🔄 리셋 점수 초기화 + 상태를 "대기"로 되돌림 (이중 확인). 오심·재경기 시 사용
코트 재배정 ↔️ 코트변경 코트 고장/조명 문제 시 다른 코트 번호로 변경

8. STEP 7 — 코트 점수판 운영

코트별 태블릿에서 /court 페이지를 열어 실시간 점수를 입력합니다.

8-1. 코트 URL 설정

코트별 URL (대회 ID = 1 기준)
코트1: /court?tid=1&court=1&locked=1 코트2: /court?tid=1&court=2&locked=1 코트3: /court?tid=1&court=3&locked=1 코트4: /court?tid=1&court=4&locked=1 코트5: /court?tid=1&court=5&locked=1 코트6: /court?tid=1&court=6&locked=1
URL 파라미터값설명
tid대회 ID (숫자)해당 대회의 경기를 표시
court코트 번호해당 코트에 배정된 경기만 표시
locked=1필수 권장다른 코트로 실수 이동 방지
autonext=true권장경기 완료 후 다음 경기 자동 로드
readonly=1선택관람 전용 모드 (점수 입력 불가)

8-2. 점수 입력 흐름 (터치)

① 대기 화면 ② 사이드 선택 ③ 점수 입력 ┌───────────────┐ ┌──────────────────┐ ┌──────────────────┐ │ 다음 경기 │ → │ 왼쪽: A팀 │ → │ 12 : 8 │ │ 대기중... │ │ 오른쪽: B팀 │ │ [좌 터치] [우 터치] │ └───────────────┘ └──────────────────┘ └──────────────────┘ │ ⑥ 대기 → 다음경기 ⑤ 서명 캡처 ④ 코트 교체 ┌──────────────┐ ┌──────────────┐ ┌──────────────────┐ │ 5초 후 │ ← │ 승팀 서명 │ ← │ 🔄 코트 교체! │ │ 다음 경기 로드 │ │ 패팀 서명 │ │ (사이드 스왑) │ └──────────────┘ └──────────────┘ └──────────────────┘
동작설명
좌/우 터치해당 영역 팀 +1점 (화면 절반씩 터치 영역)
↩ 되돌리기마지막 점수 취소 (Undo). 오입력 즉시 정정
🔄 교체코트 사이드 교체 (좌우 팀 위치 스왑). 중간점 도달 시 자동 표시
QR 코드현재 코트 관람용 URL 생성 (관중 안내)
경기 완료목표점 도달 시 자동으로 완료 → 서명 화면 표시

8-3. 코트 교체 & 서명

🔄 코트 교체 (배드민턴 / 테니스 공통)
스포츠교체 시점교체 동작
🏸 배드민턴 목표점의 절반 도달 시 (25점→13점, 21점→11점) 자동 교체 알림 + 좌우 스왑
🎾 테니스 매 2게임 (체인지오버) 자동 교체 알림 + 좌우 스왑
✍️ 서명 확인

경기 종료 시 승팀 → 패팀 순서로 서명합니다. 서명은 캡처되어 서버에 저장되므로 분쟁 시 증거로 활용할 수 있습니다.

서명 완료 후 autonext=true이면 5초 후 자동으로 다음 경기를 로드합니다.

9. STEP 8 — 결선 토너먼트

조별 리그 예선이 완료되면, 상위 팀을 결선 싱글 엘리미네이션 토너먼트로 진출시킵니다.

결선 생성 절차

1
경기 탭에서 "결선 토너먼트 생성" 버튼 클릭
2
모달에서 조별 상위 진출 팀 수 선택 (1팀 / 2팀 / 3팀)
3
같은 클럽 대결 회피 (시드 배정) 옵션 설정
4
종목별 조별 현황 미리보기에서 각 조의 순위, 승패, 진출 여부 확인
5
"결선 토너먼트 생성" 버튼 클릭 → 확인 → 결선 경기 자동 생성
📊 결선 토너먼트 특징
항목설명
방식싱글 엘리미네이션 (지면 탈락)
시드 배정크로스 시드 (1조 1위 vs 2조 2위, 1조 2위 vs 2조 1위)
BYE 처리팀 수가 2의 거듭제곱이 아닌 경우 자동 BYE 배정
클럽 회피옵션 활성화 시 같은 클럽 팀이 초반에 만나지 않도록 시드 조정
💡 결선 타이밍 팁
  • 모든 종목 예선 80% 이상 완료 시점부터 완료된 종목부터 순차 결선 가능
  • 진행률 미달 종목은 모달에서 경고 표시됨
  • 결선 생성 후에도 예선 미완료 경기는 계속 진행 가능

10. 결과 확인 & PDF 출력

대회 상세 → "결과" 탭에서 전체 순위를 확인합니다.

섹션내용

11. 보조 도구 활용

11-1. 내 경기 조회 (/my)

👤 참가자 전용 페이지

/my?tid=대회ID — 참가자가 이름을 입력하면 본인의 전체 경기 일정과 결과를 확인할 수 있습니다.

  • 경기별 코트, 상대팀, 점수, 승패 표시
  • 동명이인이 있으면 소속·급수·연도로 구분하는 선택 UI 자동 표시
  • 참가자 개인별 종합 전적 (승률, 득실차 등)
  • QR 코드를 접수처에 부착하면 스마트폰으로 바로 접속

11-2. 통계 대시보드 (/dashboard)

📊 실시간 전체 현황

/dashboard?tid=대회ID — 전광판/관중 모니터용. 30초마다 자동 갱신됩니다.

  • 전체 진행률 (진행중/대기/완료 경기 수)
  • 종목별 진행 현황 (프로그레스 바)
  • 코트별 현재 경기 상태
  • 클럽별 전적 통계 (승/패/승률)
  • 참가자 현황 (총원/남녀/체크인/참가비)

11-3. 코트 타임라인 (/timeline)

📈 경기 흐름 시각화

/timeline?tid=대회ID — 코트별로 경기가 시간순으로 배치된 타임라인 뷰. 20초마다 자동 갱신됩니다.

운영진이 어느 코트가 빠르게 진행되고 있는지, 어디가 지연되는지 한눈에 파악할 수 있습니다.

11-4. 인쇄 센터 (/print)

🖨️ 수기 운영 대비 6종 인쇄물

/print?tid=대회ID — 시스템/인터넷 장애 시 수기로 운영할 수 있도록 인쇄물을 제공합니다.

#인쇄물내용
①참가자 명단클럽별 그룹, 이름/성별/급수/연락처, 체크인·참가비 체크박스
②팀 편성표종목별 팀 목록 (선수1·선수2·급수·소속), 조 번호
③조별 대진표라운드/코트/팀A vs 팀B, 승자 체크박스
④점수 기록지코트별 경기 기록 용지 (세트 점수, 승자, 서명란)
⑤순위 집계표조별 승/패/승점/득실차/순위 기록란
⑥결선 대진표토너먼트 브래킷 (크로스 시드, BYE, 승자 흐름)

팁: 대회 전날 대진표 생성 후 인쇄해두면 비상 시 즉시 사용 가능

11-5. 푸시 알림

🔔 경기 시작 자동 알림

참가자가 /my 페이지에서 알림을 구독하면:

  • 경기 시작 시: "경기 시작!" 알림 (코트 번호, 상대 팀 포함)
  • 다음 경기 준비: 같은 코트 대기 경기 선수에게 "다음 경기 준비" 알림

  • 안드로이드 크롬: 완벽 지원 / iPhone: "홈 화면에 추가" 후 지원 (iOS 16.4+)
